Projects

I've worked on frontend dev, machine learning, exploratory data analysis, Shiny app development, and a full stack chrome extension.

Public Editor Frontend

Frontend development for PublicEditor.io, a vetted newsreading platform. Display logical fallacies and misinformation flags (collectively called "tags") as highlights on article text using jQuery, DOM API, and a specially designed FlexArray object. Visualize tag data in an interactive, recursively powered sunburst graph using a tree data structure combined with D3.js. Further work includes building a working backend with Flask and constructing Public Editor's article search engine.

Frameworks, Languages & Libraries: HTML, CSS, Javascript, Python

COVID-19 Exploratory Data Analysis

Using COVID-19 data from the Center for Systems Science and Engineering at Johns Hopkins, we explored the correlations between class and socioeconomic status with COVID-19 incidence and mortatlity. Some highlights of our exploration are: 1)

  • Binned plotting function visualizing time series incidence and mortality data binned by various social factors
  • PCA and dimensional reduction to identify leading factors in COVID incidence
  • A moving window regression model predicting COVID-19 incidence and mortality rates in all 50 states

    • Frameworks, Languages & Libraries: Python, Scikit-learn

    • Check it out on GitHub

    Site Reliability Chrome Extension

    Simple chrome extension built as a full-stack proof-of-concept. Reads your current URL using Chrome API and sends to a Flask server, which returns a information-reliability score based on a simple algorithm.

    Frameworks, Languages & Libraries: Python, Javascript, HTML, CSS, Flask
